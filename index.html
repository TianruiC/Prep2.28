The difference between D3 v4 and v5 for csv file

The difference between D3 v4 and v5 for csv is the way they handle the error situation.
D3 v5 uses premises rather than asynchronous callbacks to load data which allows users to simplify the structure of the asynchronous code.

Instead of using d3.csv("file.csv", function(error, data){if (error){console.log(error);}else {console.log(data);}};
We use d3.csv("file.csv").then(function(data){console.log(data);}); to load the data.
So we don't need to throw the error since the premise will do it automatically.

By using const data = await d3.csv("file.csv");
               console.log(data); , we can make the code even simpler.
